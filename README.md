# A reinforcement learning demo on the cliff walking grid world

(Still under construction)...

You can see it running here: <https://dominicprior.github.io/rl/>

Or you can run it locally by cloning this repo and typing vite (assuming you have installed
[Vite](https://vite.dev/)).

The original version came from asking Gemini this:

> Could you write for me a simple reinforcement learning demo to run in a browser?
> Maybe something like one of those grid worlds with a cliff. And, if possible,
> could it include a way to see the difference between Monte Carlo, Sarsa and Q-learning?

Here is Gemini's reply:

> To illustrate the differences between Monte Carlo (MC), SARSA, and Q-Learning,
> the "Cliff Walking" grid world is the perfect sandbox. In this environment,
> the agent must navigate from a start point to a goal while avoiding a "cliff"
> that resets the agent and gives a heavy penalty.

Here is the original [Gemini conversation](https://gemini.google.com/share/81b2d64e5c8d),
including the differences between Q-Learning and Sarsa.
